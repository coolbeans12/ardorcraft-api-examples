## Try the examples included in the API ##

  * [Run Simple Example!](http://ardorlabs.se/ArdorCraftAPIExample/ArdorCraftAPIExampleSimple.jnlp) - Simplest example possible. [(Source)](https://code.google.com/p/ardorcraft-api-examples/source/browse/trunk/src/com/ardorcraft/examples/simple/SimpleGame.java)
  * [Run Intermediate Example!](http://ardorlabs.se/ArdorCraftAPIExample/ArdorCraftAPIExampleIntermediate.jnlp) - Adding some more terrain generation, player collision and block editing. [(Source)](https://code.google.com/p/ardorcraft-api-examples/source/browse/trunk/src/com/ardorcraft/examples/intermediate/IntermediateGame.java)
  * **[Run Advanced Example!](http://ardorlabs.se/ArdorCraftAPIExample/ArdorCraftAPIExampleReal.jnlp)** - Select terrain generation method, texture and view distance. Set various types of blocks. Change day/night lighting. Try mesh voxelization. Walk/Fly etc. [(Source)](https://code.google.com/p/ardorcraft-api-examples/source/browse/trunk/src/com/ardorcraft/examples/thegame/RealGame.java)
